## České univerzity

ČVUT FIT — Computer Security (navazující Mgr.)
* Program a plán: přehled specializace + studijní plán, včetně seznamu předmětů. ([fit.cvut.cz][1])

Masarykova univerzita FI — Computer Systems, Communication and Security (Mgr.), specializace Information Security
* Katalog specializace a typické povinné/volitelné předměty (Applied Cryptography, Secure Coding, Secure Network Design…). ([fi.muni.cz][2])
* Detailní studijní plán (obligatory courses). ([Informační systém MU][3])

VUT Brno — FIT: Information Technology Security; FEKT: Information Security (Mgr.)
* FIT: profil oboru a cíle, přehled bezpečnostních kurzů. ([fit.vut.cz][4])
* FEKT: program Information Security (popis, výstupy absolventa). ([utko.fekt.vut.cz][5])

VŠB–TUO — kyberbezpečnost (vybrané předměty)
* Cyber Security I/II (okruhy: zabezpečení sítí, moderní šifry, trendy; anotace v e-výuce). ([vsb.cz][6])

ZČU (FAV/KIV) — bezpečnostní kurzy
* KIV/BIT (Computer Security/Bezpečnost IT) a KIV/BSA (Security of Network Applications) — anotace, sylabus, studijní materiály. ([portal.zcu.cz][7])

UK MFF — Information Security I/II (v rámci softwarových systémů)
* Přehled studijního průvodce se seznamem předmětů + karty kurzů NSWI089/NSWI071. ([mff.cuni.cz][8])

## Zahraničí

MIT — 6.858 Computer Systems Security; 6.857 Network and Computer Security
* Sylaby a materiály (přednášky, laby, témata: OS, web, hardware, capabilities…). ([MIT: Computer Systems Security](https://ocw.mit.edu/courses/6-858-computer-systems-security-fall-2014/pages/syllabus/))

University of Oxford — MSc in Software and Systems Security
* Oficiální popis programu + modulová struktura a informační PDF (2025/26 entry). ([Univerzita Oxford: MSc in Software and Systems Security - Courses](https://www.ox.ac.uk/admissions/graduate/courses/msc-software-and-systems-security))

UCL — MSc Information Security
* Stránka programu (core: computer security, cryptography, secure comms; volitelné moduly). ([University College London: Information Security MSc](https://www.ucl.ac.uk/prospective-students/graduate/taught-degrees/information-security-msc))

Royal Holloway, University of London — MSc Information Security / MSc Cyber Security
* Přehled kurikula a modulová specifikace (kredity, jádro/volitelné, aktualizace 2023/24). ([Royal Holloway: Information and Cyber Security MSc](https://www.royalholloway.ac.uk/studying-here/postgraduate/information-security/information-security/))

University of Cambridge — Software and Security Engineering (kurz, materiály)
* Rozpis přednášek a studijní materiály (policy, protokoly, usable security, risk). ([Katedra informatiky na Cambridgi: Course pages 2020–21: Software and Security Engineering](https://www.cl.cam.ac.uk/teaching/2021/SWSecEng/))
