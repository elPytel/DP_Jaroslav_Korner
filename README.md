# Diplomová práce


## Zadání

Příprava interaktivních úloh z počítačové bezpečnosti pro vysokoškolské studenty

5 bodů zadání:
- Rešerše stávajících řešení na trhu 
- Analýza potřeb cílové skupiny 
- Návrh souboru praktických úloh 
- Implementace a testování úloh v laboratorním prostředí 
- Vyhodnocení úloh na skupině studentů
